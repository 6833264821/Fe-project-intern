## Fordev

Setup ก่อนเริ่มพัฒนา

1. ติดตั้ง Node.js LTS (แนะนำเวอร์ชัน 20 ขึ้นไป)
2. ติดตั้ง Angular CLI เวอร์ชัน 18
	- npm install -g @angular/cli@18
3. ติดตั้ง dependencies ของโปรเจกต์
	- npm install

PrimeNG ที่โปรเจกต์นี้ใช้

1. primeng@18
2. @primeuix/themes
3. primeicons
4. primeflex
